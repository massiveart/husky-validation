Form Options

Name Default Description
submitEvent true avoid submit if not valid
element.trigger focusout default validate trigger for each element
element.addclasses true add error and success classes for each element
{% highlight javascript %} define(['js/validation'], function(Validation) { var validation = new Validation($('#contact-form'), { submitEvent: false, element: { trigger: 'keyup', addclasses: false } }); }); {% endhighlight %}

Field Options

Name Option Default Description
trigger data-trigger focusout Validate trigger
addclasses data-addclasses true add error and success classes
validate data-validate true If set the field will be validated, if not the field will be ignored

Built-In Validators

Name API Description
Required data-required="true" Validate that a required field has been filled.
Max data-max="100" Validates that a given number is less than some maximum number.
Min data-min="20" Validates that a given number is greater than some minimum number.
Min Length dta-minlength="3" Validates that the length of a string is at least as long as the given limit.
Max Length data-maxlength="10" Validates that the length of a string is not larger than the given limit.

Built-In Types

Name API Description
Date data-type="date" Validates that a value is a valid ISO date.
Decimal data-type="decimal" Validates that a value is a valid number.
Email data-type="email" Validates that a value is a valid email address.
Url data-type="url"
data-url-strict="true" 		Strict=false: Validates that a value is a valid url.
Strict=true: Validates that a value is a valid strict URL, ie: with one of http, https, ftp, git allowed protocols.

JavaScript API

Validation Object

Name Return Description Example
Create Object Bind Validation to a form var validation = new Validation($('#contact-form'));
Get Object Get the Object for form var validation = $('#contact-form').data('validation')
Validate Boolean Revalidate Form validation.validate()
isValid Boolean Get last Validation result validation.isValid()
Add Constraint Add a Constraint to a field validation.addConstraint('#field', 'minlength', {minlength: 6})
Update Constraint Update a Constraint from a field validation.updateConstraint('#field', 'minlength', {minlength: 6})
Delete Constraint Delete a Constraint from a field validation.updateConstraint('#field', 'minlength')

Element Object

Name Return Description Example
Get Object Get the element for field var element = $('#field').data('element')
Validate Boolean Revalidate Field element.validate()
Add Constraint Add a Constraint to a field element.addConstraint('minlength', {minlength: 6})
Update Constraint Update a Constraint from a field element.updateConstraint('minlength', {minlength: 6})
Delete Constraint Delete a Constraint from a field element.updateConstraint('minlength')
